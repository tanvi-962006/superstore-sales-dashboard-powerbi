# 📊 Superstore Sales Dashboard — Power BI

An interactive Power BI dashboard analyzing **9,800+ sales transactions** from the Superstore dataset to uncover key business insights across regions, products, and customer segments.



---

## 🎯 Objective

Built a **3-page interactive dashboard** to help stakeholders:
- Track sales performance and growth trends
- Identify top-performing regions, products, and customers
- Analyze category-wise and segment-wise revenue distribution
- Make data-driven business decisions

---

## 📈 Key Insights

| Insight | Detail |
|---|---|
| 💰 Total Sales | **$2.26M** across 4 years |
| 📦 Total Orders | **4,922** unique orders |
| 👥 Total Customers | **793** unique customers |
| 🏆 Top Region | **West** leads in sales |
| 🖥️ Top Category | **Technology** (36.59%) |
| 👤 Top Customer | **Sean Miller** (~$25K) |
| 📊 Top Segment | **Consumer** (50.76%) |
| 📈 Growth Trend | Sales grew consistently from 2015 ($0.48M) to 2018 ($0.72M) |

---

## 📄 Dashboard Pages

### Page 1: Executive Overview
- 4 KPI Cards (Total Sales, Orders, Customers, Avg Order Value)
- Sales Trend (Line/Area Chart)
- Sales by Category (Donut Chart)
- Sales by Region (Bar Chart)

### Page 2: Regional & Geographic Analysis
- State-wise Sales Heatmap (Filled Map)
- Region × Category Breakdown (Stacked Bar Chart)
- State-wise Sales Table with Data Bars

### Page 3: Product & Customer Deep Dive
- Category → Sub-Category Hierarchy (Treemap)
- Top 10 Customers (Bar Chart)
- Sales by Segment (Donut Chart)
- Ship Mode × Category Breakdown (Stacked Bar)

## 🛠️ Tools & Skills Used

- **Power BI Desktop** — Data modeling, DAX, Visualizations
- **Power Query** — Data cleaning & transformation
- **DAX Measures** — KPIs, YoY Growth, Calculated metrics
- **Data Visualization** — Charts, Maps, Conditional Formatting
- **UI/UX** — Dark theme, consistent color palette, cross-filtering

---

## 📂 Dataset

- **Source**: [Superstore Sales Dataset — Kaggle](https://www.kaggle.com/datasets/rohitsahoo/sales-forecasting-dataset)
- **Records**: 9,800+ rows
- **Columns**: Order ID, Order Date, Ship Mode, Segment, Region, Category, Sub-Category, Sales, etc.
- **Time Period**: 2015–2018

---

## 🚀 How to Use

1. Download the `.pbix` file from this repository
2. Open in [Power BI Desktop](https://powerbi.microsoft.com/desktop/) (free)
3. Interact with slicers and cross-filter by clicking on any chart element
4. Navigate between pages using the bottom tabs or arrow buttons
